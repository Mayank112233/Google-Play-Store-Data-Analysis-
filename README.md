# Google Play Store Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Google Play Store data to identify trends in app ratings, reviews, installs, pricing, and categories.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified top-performing app categories.
- Analyzed relationships between ratings and installs.
- Explored pricing patterns and user engagement.
- Generated visualizations for business insights.

## Dataset
Google Play Store Dataset
